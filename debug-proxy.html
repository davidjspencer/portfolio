debugProxy

UI console for debugging

UIBrandingWeb design

The challenge

The challenge was to create an interface that allows you to view, pause and modify traffic sent through the proxy server. A collaboration with a software engineer, debugProxy is a HTTP/S proxy server that can be used by any device that supports using HTTP Proxy servers. I only worked on the front-end.

The Solution

I broke the different parts of the process into components and unified the components using gestalt principles (laws of proximity, common region, focus and similarity). I designed the UI to make a complex process intuitive for the user.

The design resembles a command line interface and text editors used for programming. The colour scheme is called Solaris, created by Ethan Schoonover, which is a favourite of developers. The main typeface is Inconsolata, a monospaced font designed for code.
